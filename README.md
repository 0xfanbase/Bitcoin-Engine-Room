# BTC Engine Room

A free, public, self-auditing Bitcoin fundamentals + price-model dashboard. Static site on GitHub Pages, daily self-healing data pipeline on GitHub Actions, zero backend, zero running cost.

**Status: Phase 1 — Skeleton & backfill.** See `PROGRESS.md` for the session log, `CLAUDE.md` for project conventions and hard rules, and `docs/BTC_ENGINE_ROOM_BUILD_SPEC.md` for the full build spec this project follows.

Data source attribution, disclaimers, and the live dashboard land in later phases (see `IMPROVEMENT_BACKLOG.md` and the phase checklist in `PROGRESS.md`).
